Detailed Description

Dataholder used for make command creation.

By issuing:

MakeParameter.BUILDDIR.name # -> BUILDDIR
# or
MakeParameter.BUILDDIR.value # -> "/dev/shm/amiroCI"

the single parts of the make command can be assembled.
